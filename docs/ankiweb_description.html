GitHub link One drawback to the otherwise powerful Anki template editor is the absence of two common features of text editors:
  1. fixed width (monospace) text
  2. syntax highlighting
A common ad-hoc workaround was to copy & paste between the Anki template editor and a separate text editor. This addon provides these missing features directly in the template editor. Syntax highlighting works for both HTML and CSS, although currently the lexing is naive. Example: Next steps:
  1. configurable highlighting (colors, tokens)
  2. better syntax highlighting
  3. line numbers
  4. search & replace
  5. tabs to spaces
  6. automatic indentation
